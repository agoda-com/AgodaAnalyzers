Use await Task.WhenAny/All instead of Task.WaitAny/All

To wait for tasks to finish, use await Task.WhenAll/Any instead of Task.WaitAll/Any. The latter blocks the thread while the tasks complete, defeating the purpose of using async.

Noncompliant Code Example

var task1 = DownloadFileAsync("..."); var task2 = DownloadFileAsync("..."); Task.WaitAll(task1, task2);

Compliant Code Example

var task1 = DownloadFileAsync("..."); var task2 = DownloadFileAsync("..."); await Task.WhenAll(task1, task2); // does not block
