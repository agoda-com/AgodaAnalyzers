If `public` class/interface method/property return value implements `IEnumerable`, then it can _only_ be declared as one of the following: - `IEnumerable` - `ISet` - `IList` - `IDictionary` - `IReadOnlyDictionary` - `KeyedCollection` - `byte[]` (special case for raw binary data) - `string` (which happens to implement `IEnumerable`)

Noncompliant Code Example

    public List GetStrings() { ... }
    public List GetStringsV2 { get; set; }

Compliant Code Example

    public IList GetStrings() { ... }
    public IList GetStringsV2 { get; set; }
